Ezgif.com is a simple, free online GIF maker and toolset for basic animated image editing.
Here you can create, edit and convert GIF, APNG, WebP, MNG and AVIF animations.

ezgif.com news and updates

  • Dec 21, 2024

    Fixed an issue where APNG files produced by EZGIF were incompatible with some other software and websites.

  • Dec 16, 2024

    The GIF Combine tool can now join files not only side by side but also one on top of the other. Added MP4 to MP3 converter.

  • Dec 11, 2024

    Added play/pause button (also works with spacebar) to the crop video tool.
    New converters: WebP to SVG and SVG to WebP
    Added quality settings to JPG to WebP and PNG to WebP converters.

  • Jul 9, 2024

    Added 16 FPS option to video to GIF/APNG/WebP converters.

  • Mar 6, 2024

    Added PNG to SVG and JPG to SVG converters using Autotrace and VTracer tools.

  • Feb 23, 2024

    We have doubled the maximum upload file size! Now you can upload GIFs and other files up to 200MB.

    The maximum image resolution in the GIF maker and resizer tools has been increased from 1200x1200px to 1920x1920px to allow for the creation of Full HD (FHD) GIFs.

  • Feb 6, 2024

    Added WebP to WebM and APNG to WebM converters.
    Improved support for animated WebP files with transparent backgrounds.

    Support for the FLIF image format has been removed. Development of this format appears to have been abandoned for several years, and it never achieved widespread adoption. Additionally, some of the FLIF tools have been broken for a while, and I don't feel it would be worth the effort to fix them.

  • Jan 17, 2024

    • Created a dedicated and improved tool for background removal.
    • Added Telegram sticker to animated GIF converter.
    • Added a dropdown menu under the uploaded image to allow access to less commonly used tools.

  • Jan 13, 2024

    Now you can upload images to EZGIF directly from the clipboard by pressing Ctrl/Command+V on any of the tool pages.

  • Jan 11, 2024

    Support for the JPEG XL (.jxl) file format has been added. You can now create, convert, and edit animated JXL files using most of the EZGIF tools.

  • Jan 7, 2024

    Following requests from some users, from now on, output files saved from EZGIF.com will be named starting with the original filename.

  • Jan 2, 2024

    A few recent updates:

    • Improved drag-and-drop support in upload forms
    • Improved AVIF support in several tools
    • Added a 3GP to MP4 converter
    • Introduced an option to view file metadata
